# 📧 Smart Email Sorting Agent

An **NLP-powered AI agent** that classifies incoming emails into categories like **Work, Personal, and Spam**.  
Built using **TF-IDF + Logistic Regression**, and deployed with **Chainlit** for an interactive UI.  

---

## 🚀 Features
- Email text preprocessing with **NLTK** (stopwords, tokenization, stemming).
- **TF-IDF feature extraction** for text representation.
- **Logistic Regression model** trained on Kaggle email dataset.
- Real-time classification via **Chainlit interface**.
- Saves trained model and vocabulary for reuse.
